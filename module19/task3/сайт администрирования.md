### Создание объектов в базе данных Django

Используя Django shell

#### Создание объектов

```
from myapp.models import User, Post, Category, Comment, Tag

# Создание пользователей
john = User.objects.create(username='john_doe', email='john@example.com', password='hashed_password')
jane = User.objects.create(username='jane_doe', email='jane@example.com', password='hashed_password')

# Создание категорий
tech = Category.objects.create(name='Technology', description='Posts related to technology')
lifestyle = Category.objects.create(name='Lifestyle', description='Posts related to lifestyle')

# Создание тегов
python_tag = Tag.objects.create(name='Python')
django_tag = Tag.objects.create(name='Django')

# Создание постов
post1 = Post.objects.create(title='My first post', slug='my-first-post', body='This is the body of my first post', author=john, category=tech)
post2 = Post.objects.create(title='Another post', slug='another-post', body='This is the body of another post', author=jane, category=lifestyle)

# Создание комментариев
comment1 = Comment.objects.create(body='Great post!', author=jane, post=post1)
comment2 = Comment.objects.create(body='Nice article!', author=john, post=post2)

# Связь постов и тегов
post1.tags.add(python_tag, django_tag)
post2.tags.add(python_tag)
```

### Изменение одного из элементов статьи

```
# Изменение заголовка статьи
post1.title = 'Updated first post title'
post1.save()
```

### Получение всех объектов из базы данных

```
# Получение всех постов
all_posts = Post.objects.all()
print(all_posts)
```

### Удаление одного из объектов базы данных

```
# Удаление поста
post2.delete()
```

### Фильтрация по объектам в базе данных

```
# Фильтрация статей опубликованных в 2024 году
posts_2024 = Post.objects.filter(publish__year=2024)
print(posts_2024)

# Фильтрация статей, созданных пользователем admin
admin_posts = Post.objects.filter(author__username='admin')
print(admin_posts)
```
